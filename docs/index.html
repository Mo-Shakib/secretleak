How it works

From install to protected in 60 seconds

secretleak plugs into your existing workflow with zero configuration.

1

Install

Install via pip and you're ready to scan immediately.

2

Scan

Point secretleak at any directory or Git repo. Supports working tree, staged changes, and commit ranges.

3

Review

Rich console table with severity, masked secret, and a clickable file link to the exact line.

4

Block

Install the pre-commit hook to automatically prevent secrets from ever being committed.

BASH — Scan modes secretleak
# Scan a directory $ secretleak scan ./my-project/ # Scan only staged changes $ secretleak scan --staged # Scan a commit range $ secretleak scan --commit-range HEAD~10..HEAD # Output JSON report $ secretleak scan --format json -o report.json # Output SARIF for GitHub $ secretleak scan --format sarif -o results.sarif
YAML — .secretleak.yaml config
# Tune entropy detection entropy: enabled: true threshold: 4.5 min_length: 20 # Suppress known false positives allowlist: patterns: - "EXAMPLE" - "placeholder" paths: - "**/tests/**" # Extra ignore paths ignore_paths: - "**/fixtures/**"

🔬 Shannon Entropy Detection

Catches secrets that don't match any known pattern by measuring the randomness of strings. High entropy ≈ likely secret.

Normal text1.8 bits/char
Filesystem path3.2 bits/char
Base64 secret (threshold)4.5 bits/char ←
Random API key5.8 bits/char

Regex matches suppress overlapping entropy matches — regex always wins to minimise false positives.

Installation

Up and running in seconds

Requires Python 3.11+. No external services, no sign-up, no telemetry.

Install from PyPI
$ pip install secretleak # Or install with dev extras $ pip install "secretleak[dev]"
Install from source
$ git clone https://github.com/Mo-Shakib/secretleak $ cd secretleak $ pip install -e ".[dev]"
Quick start
# Scan your current project $ secretleak scan . # Install pre-commit hook $ secretleak install-hook # Generate baseline (suppress known findings) $ secretleak generate-baseline . \ -b .secretleak-baseline.json # List all active rules $ secretleak rules # Show version $ secretleak --version
